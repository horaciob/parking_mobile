=== README
tested on ruby 2.2.1p85

=== Dependencies

Mysql

=== To install
git clone git@github.com:horaciob/parking_mobile.git

cd parking_mobile/webserver/

bundle install --path vendor/bundle

vim config/database.yml #Set up your mysql or wathever database that you want to use
rake db:create:all

rake db:migrate

** if all was ok you will be able to run test :)

rspec

should run test and if all pass.

puma -C config/puma.rb

or

rails s Puma

curl localhost:3000
